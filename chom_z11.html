1. Selektory proste
    1. p
      {
      color:red;
      font-size:14pt;
      }
    2. Selektorem prostym jest znaczniki języka XHTML/HTML (np. p, h3, table, img)
      przypisany do elementu, który chcesz w danym momencie formatować.
  2. selektor uniwersalny
    1. * { cecha: wartość }
    2. Selektor taki pozwala ustalić określone atrybuty dla wszystkich elementów strony
      , a więc dla różnych selektorów typu
  3. selektor potomka
    1. przodek1 przodek2 ... potomek { cecha: wartość }
    2. Przy użyciu selektora potomka można formatować elementy, które sa zawarte wewnątrz
      innych znaczników, czyli leżą niżej w hierarchii drzewa dokumentu.
  4. selektor dziecka
    1. rodzic > dziecko {właściwość: wartość}
    2. Selektor dziecka służy do definiowania elementów, które znajdują się o jeden
      rząd niżej w hierarchii drzewa dokumentów.
  5. selektor braci
    1. brat1 + brat2 {właściwość: wartość}
    2. Zastosowanie selekora rodzeństwa można przypisać styl elementowi bezpośrednio
      sąsiadującemu z innymi. By warunek został spełniony, oba selektory muszą mieć wspólnego
      „rodzica”.
  6. selektor atrybutu
    1. selektor[atrybut] { cecha: wartość }
    2. Polecenie to może zostać wykorzystane dla elementów, którym z poziomu (X)HTML został
      nadany określony atrybut (przy czym jego wartość nie ma znaczenia).
